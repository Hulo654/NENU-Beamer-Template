# NENU LaTeX Beamer Theme 模板
一个基于 Berlin 修改而成的 Beamer 模板，东北师范大学 NENU 学术汇报模板。
下载所有"代码"或打开 berlinsun.tex 文件后，即可看到模板的源代码，复制粘贴即可使用。
